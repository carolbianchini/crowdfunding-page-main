### O Que Aprendi

Ao realizar o projeto aprendi mais sobre CSS Grid e layout responsivo.

Segue exemplo, trechos dos códigos abaixo: 

CSS
```css
    .input-box {
    display: grid;
    grid-template-columns: repeat(5, 1fr);
    grid-template-rows: repeat(5, 1fr);
    grid-column-gap: 0px;
    grid-row-gap: 0px;
    }
```
